Dialog

A generic alert, confirm and flexible dialog with optional overlay.

dialog({
  type: 'custom' // tips | confirm
, title: 'Displayed at the top of the dialog box.'
, content: 'Plain text or HTML strings.'
})
.setButtons(customButtonsHTMLString)
.on('confirm', function() {
  // Do something
})
.open()

Instance Methods

Dialog.prototype.setTitle()

Overwrites current title of the dialog box.

dialog().setTitle('New title')
Dialog.prototype.setFoot()

Adds bottom section to the dialog box.

dialog().setFoot('<input type="checkbox"> Agree')
Dialog.prototype.setButtons()

Sets buttons text or style on demand.

dialog({
  type: 'tips'
})
.setButtons([{
  text: '知道了'
}])
Dialog.prototype.addClass()

Adds classname to #ark-dialog

dialog().addClass('author-dialog')
Dialog.prototype.open()

Shows your dialog box.

dialog().open()
Dialog.prototype.close()

Removes your dialog box.

dialog().close()
